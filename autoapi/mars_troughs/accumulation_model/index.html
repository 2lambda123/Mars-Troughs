class mars_troughs.accumulation_model.InsolationAccumulationModel(times: numpy.ndarray, insolations: numpy.ndarray)¶

Bases: AccumulationModel

An accumulation rate model that depends on solar insolation, A(Ins(t)). A is in m/year. Interpolated splines are created for the insolation as a function of time for faster integration.

Parameters

  • times (np.ndarray) – times at which the solar insolation is known (in years)

  • insolation (np.ndarray) – values of the solar insolation (in W/m^2)

get_xt(self, time: numpy.ndarray, int_retreat_model_t_spline: numpy.ndarray, cot_angle, csc_angle)¶

Calculates the horizontal distance x (in m) traveled by a point in the center of the high side of the trough. This distance x is a function of the accumulation rate A(ins(t)) and the retreat rate of ice R(l(t),t) as in dx/dt=(R(l(t),t)+A(ins(t))cos(theta))/sin(theta). Where theta is the slope angle of the trough.

Parameters

time (np.ndarray) – times at which we want the path.

Output:

horizontal distances (np.ndarray) of the same size as time input, in meters.

class mars_troughs.accumulation_model.LinearInsolationAccumulation(times: numpy.ndarray, insolations: numpy.ndarray, intercept: float = 1e-06, slope: float = 1e-06)¶

Bases: InsolationAccumulationModel, mars_troughs.generic_model.LinearModel

Accumulation is linear in solar insolation. A(ins(t)) = intercept + slope*ins(t). A is in m/year.

Parameters

  • times (np.ndarray) – times at which the solar insolation is known (in years)

  • values (insolation) – values of solar insolation (in W/m^2)

  • intercept (float, optional) – accumulation rate at present time. Default is 1e-6 m/year

  • slope (float, optional) – default is 1e-6 m/year per unit of solar insolation (m^3/(year*W)).

get_yt(self, time: numpy.ndarray)¶

Calculates the vertical distance y (in m) traveled by a point in the center of the high side of the trough. This distance is a function of the accumulation rate A as y(t)=integral(A(ins(t)), dt) or dy/dt=A(ins(t))

Output:

np.ndarray of the same size as time input containing values of the vertical distance y, in meters.

class mars_troughs.accumulation_model.QuadraticInsolationAccumulation(times, insolation, intercept: float = 1.0, linearCoeff: float = 1e-06, quadCoeff: float = 1e-06)¶

Bases: InsolationAccumulationModel, mars_troughs.generic_model.QuadModel

Accumulation rate A (in m/year) as a quadratic polynomial of insolation. A(ins(t)) = intercept + linearCoeff*ins(t)+ quadCoeff*ins(t)^2. A is in m/year.

Parameters

  • times (np.ndarray) – times at which the solar insolation is known, in years.

  • insolations (np.ndarray) – value of the solar insolations (in W/m^2)

  • intercept (float, optional) – default is 1 m/year

  • linearCoeff (float, optional) – default is 1e-6 m/year per unit of solar insolation (m^3/(year*W)).

  • quadCoeff (float, optional) – default is 1e-6 m/year per unit of solar insolation squared (m^5/(year*W^2)).
