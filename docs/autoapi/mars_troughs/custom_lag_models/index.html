Abstract class for lag models, that have a method called get_lag_at_t() that returns the lag as a function of time.

prefix :str = lag_

All parameters of lag models start with ‘lag’.

get_lag_at_t(time: numpy.ndarray) → numpy.ndarray

Lag as a function of time

Parameters

time (np.ndarray) – times at which we want to calculate the lag.

Output:

np.ndarray of the same size as time input containing values of lag.

Parameters

class mars_troughs.custom_lag_models.LinearLag(constant: float = 1e-06, slope: float = 1e-08)

Bases: LagModel, mars_troughs.generic_model.LinearModel

The lag thickness is linear in time. Lag changes as lag(t) = intercept + slope*t.

Parameters

  • intercept (float, optional) – default is 1 millimeter. The lag thickness at time t=0 (present day).

  • slope (float, optional) – default is 1e-6 mm per year. The rate of change of the lag per time.
